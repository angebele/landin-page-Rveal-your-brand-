# Landing Page — Ange Belegue · Personal Branding

Landing page pour le mentor en personal branding Ange Belegue.

## Déployer sur Vercel

1. Connecte ce repo à Vercel : [vercel.com/new](https://vercel.com/new)
2. Import du repo → Deploy
3. C'est live en ~30 secondes

## Fichiers

- `index.html` — la page (tout est dedans : HTML + CSS + JS)
- `vercel.json` — config pour Vercel

## Modifier la page

Edite `index.html` directement — c'est un fichier unique, autonome, sans dépendances.

Les sections principales :
- Hero (S1)
- Problème (S2)
- Reframe (S3)
- Transformation (S4)
- Méthode (S5)
- Pourquoi moi (S6)
- Pour qui (S7)
- Témoignages (S8)
- Garantie (S9)
- FAQ (S10)
- CTA Final (S11)
